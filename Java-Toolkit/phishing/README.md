#  Phishing Message Detector 

A simple rule-based cybersecurity tool that detects phishing patterns in text messages using keyword-based analysis.  
It simulates how basic security systems identify social engineering attacks.

---

##  Objective
- Detect phishing-like messages using predefined rules
- Identify social engineering tactics such as urgency, threats, and credential theft
- Build awareness of basic cybersecurity filtering logic

---

##  Approach
- Check for:
  - Urgency-based phrases
  - Account/security threat indicators
  - Credential/action request patterns
- Flag message as phishing if any rule is triggered

---

##  Inference
- Phishing messages often rely on urgency + fear + action prompts
- Credential requests (password/login) are strong phishing indicators
- Multiple triggers increase suspicion level
- Absence of patterns does not guarantee safety (rule-based system limitation)

---

##  Limitations
- Only program inbuilt keyword-based detection

---

## Concepts Demonstrated
- Social engineering detection
- Heuristic-based intrusion detection
